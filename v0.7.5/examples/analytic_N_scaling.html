Analytic error scaling vs. number of bins¶

examples/analytic_N_scaling.py runs a series of integrations with vaired parameters, examples/plot_analytic_N_scaling.py plots the error in the solutions vs. number of bins for each set of other parameters. For example, one may expect different error scaling for the different number of stencil points used. (\(N^{-2}\), \(N^{-4}\) and \(N^{-6}\) for 3, 5 and 7 stencil points respectively)

$ python analytic_N_scaling.py --help

usage: analytic_N_scaling.py [-h] [-p] [-s SAVEFIG] [-g GEOMS] [--nNs NNS]
                             [-N NS] [--nspecies NSPECIES]
                             [--nstencils NSTENCILS] [-v]

optional arguments:
  -h, --help            show this help message and exit
  -p, --plot            False
  -s SAVEFIG, --savefig SAVEFIG
                        'None'
  -g GEOMS, --geoms GEOMS
                        'fcs'
  --nNs NNS             7
  -N NS, --Ns NS        -
  --nspecies NSPECIES   '1,2,3'
  --nstencils NSTENCILS
                        '3,5,7'
  -v, --verbose         False

$ python plot_analytic_N_scaling.py --help

usage: plot_analytic_N_scaling.py [-h] [-s SAVEFIG] [-d DPI] [-e] [--linx]
                                  [--categorical CATEGORICAL]
                                  [--continuous CONTINUOUS] [--liny]
                                  [-y YLIMS] [-n NFITS] [--prop PROP]
                                  [--prop-label PROP_LABEL]

optional arguments:
  -h, --help            show this help message and exit
  -s SAVEFIG, --savefig SAVEFIG
                        'None'
  -d DPI, --dpi DPI     300
  -e, --errorbar        False
  --linx                False
  --categorical CATEGORICAL
                        'geom,nspecies,nstencil'
  --continuous CONTINUOUS
                        'N'
  --liny                False
  -y YLIMS, --ylims YLIMS
                        'None'
  -n NFITS, --nfits NFITS
                        '7,6,4'
  --prop PROP           'rmsd_over_atol'
  --prop-label PROP_LABEL
                        'RMSD/atol'

Here is an example generated by:

$ python analytic_N_scaling.py --nNs 6
$ python plot_analytic_N_scaling.py --savefig analytic_N_scaling.png
