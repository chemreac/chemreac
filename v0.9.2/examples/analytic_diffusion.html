Analytic diffusion¶

examples/analytic_diffusion.py models a diffusion process and reports the error from the model integration by comparison to the analytic solution (intial concentrations are taken from Green’s function expressions for respective geometry).

$ python analytic_diffusion.py --help

::

$ python analytic_diffusion.py –plot –efield –center 0.5 –nstencil 5 –nspecies 3 –geom f

$ python analytic_diffusion.py –x0 0 –xend 1000 –N 1000 –center 500 -D 400 –nstencil 3

Note -D 475

$ python analytic_diffusion.py --x0 0 --xend 1000 --N 1000 --center 500 -D 475 --nstencil 7

Still problematic (should not need to be):

$ python analytic_diffusion.py --plot --nstencil 5 --logy --D 0.0005

Here is an example generated by:

$ python analytic_diffusion.py --plot --nstencil 3 --nspecies 2 --geom f --savefig analytic_diffusion.png
analytic_diffusion.cylindrical_analytic(x, t, D, mu, x0, xend, v, logy=False, logx=False, use_log2=False)[source]¶

Evaluates the Green’s function:

\[c(x, t) = \frac{x_{end}-x_{0}}{4 \pi D t} \ e^{-\frac{(x - \mu - vt)^2}{4Dt}}\]

which satisfies:

\[\frac{\partial c(x, t)}{\partial t} = D\nabla^2 \ c(x, t) - \vec{v} \cdot c(x, t)\]

where \(\nabla\) in cylindrical coordinates with axial symmetry is:

\[\nabla \cdot c = \frac{1}{x}\frac{\partial \left( x c \right)}{\partial x}\]

and where \(\nabla^2\) is:

\[\nabla^2 = \frac{1}{x} \frac{\partial}{\partial x} \left( x \frac{\partial}{\partial x} \right)\]
analytic_diffusion.flat_analytic(x, t, D, mu, x0, xend, v, logy=False, logx=False, use_log2=False)[source]¶

Evaluates the Green’s function:

\[c(x, t) = \frac{x_{end}-x_{0}}{\sqrt{4 \pi D t}} \ e^{-\frac{(x - \mu - vt)^2}{4Dt}}\]

which satisfies:

\[\frac{\partial c(x, t)}{\partial t} = D\nabla^2 \ c(x, t) - \vec{v} \cdot c(x, t)\]

where \(\nabla\) in cartesian coordinates with planar (yz-plane) symmetry is:

\[\nabla = \frac{\partial}{\partial x}\]

and where \(\nabla^2\) :

\[\nabla^2 = \frac{\partial^2}{\partial x^2}\]

analytic_diffusion.spherical_analytic(x, t, D, mu, x0, xend, v, logy=False, logx=False, use_log2=False)[source]¶

Evaluates the Green’s function:

\[c(x, t) = \frac{x_{end}-x_{0}}{\sqrt{4 \pi D t^3}} \ e^{-\frac{(x - \mu - vt)^2}{4Dt}}\]

which satisfies:

\[\frac{\partial c(x, t)}{\partial t} = D\nabla^2 c(x, t) - \vec{v} \cdot c(x, t)\]

where \(\nabla\) in spherical coordinates for an isotropic system is:

\[\nabla \cdot c = \frac{1}{x^2}\frac{\partial \left( x^2 c \right)}{\partial x}\]

and where \(\nabla^2\) is:

\[\nabla^2 = \frac{1}{x^2} \frac{\partial}{\partial x} \left( x^2 \frac{\partial}{\partial x} \right)\]
