Analytic error scaling vs. number of bins¶

examples/analytic_N_scaling.py runs a series of integrations with vaired parameters, examples/plot_analytic_N_scaling.py plots the error in the solutions vs. number of bins for each set of other parameters. For example, one may expect different error scaling for the different number of stencil points used. (\(N^{-2}\), \(N^{-4}\) and \(N^{-6}\) for 3, 5 and 7 stencil points respectively)

$ python analytic_N_scaling.py --help

::
$ python plot_analytic_N_scaling.py –help

Here is an example generated by:

$ python analytic_N_scaling.py --nNs 6
$ python plot_analytic_N_scaling.py --savefig analytic_N_scaling.png
